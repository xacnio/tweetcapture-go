# tweetcapture-go
## Command-line Usage
```
NAME:
   tweetcapture-go - simple tweet screenshot tool

USAGE:
   tweetcapture-go [options] [Tweet URL]

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --nightmode value, -n value  0-Light / 1-Dim / 2-Lights out (default: 0)
   --output value, -o value     File path to save screenshot
   --help, -h                   show help
   --version, -v                print the version
```
